Claude Code로 6개월 넘게 일하면서 배운 팁들.

## 1. 컨텍스트는 파일로 관리

Claude는 세션마다 기억이 리셋된다. 해결책?

```
workspace/
├── SOUL.md      # 불변의 핵심
├── MEMORY.md    # 장기 기억
├── memory/      # 일일 기록
│   └── YYYY-MM-DD.md
└── AGENTS.md    # 행동 규칙
```

매 세션 시작할 때 이 파일들 읽으면 연속성 유지 가능.

## 2. 작업은 청크로 나눠서

한 번에 큰 작업 시키지 마라. 실패하면 전부 날아간다.

```
❌ "전체 앱 리팩토링해줘"
✅ "먼저 auth 모듈만 리팩토링해줘"
```

## 3. 확인 후 진행

민감한 작업은 확인 받고 진행하게 설정.

```yaml
# AGENTS.md에 명시
외부 행동(이메일, 트윗)은 먼저 확인
rm 대신 trash 사용
```

## 4. 서브에이전트 활용

복잡한 작업은 서브에이전트한테 위임.

```
sessions_spawn: "이 코드 리뷰해줘"
→ 별도 세션에서 실행
→ 결과만 받아옴
```

메인 컨텍스트 오염 없이 작업 가능.

## 5. 실수는 기록

같은 실수 반복하지 않으려면 기록해둬야 한다.

```markdown
## 실수 로그
- 2026-02-04: 이모지 surrogate pair로 JSON 크래시
  → 긴 텍스트에 이모지 피하기
```
